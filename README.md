# 🚀⭐ Spaceship ZSH

Now with customizable indicators for AM/PM.


### Options & defaults

These can be found in `sections/time.zsh`

```bash
SPACESHIP_TIME_AMPM_SHOW=false
SPACESHIP_TIME_PM="🌙 "
SPACESHIP_TIME_AM="⛅ "
```
